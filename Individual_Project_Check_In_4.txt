2 FINAL PROTOTYPE
• Live demo: https://hbycwyh2008.github.io/check-in4/
• How to view: Open the URL in Google Chrome (or Microsoft Edge). No login
required. Interact with the full flow from the overview through rubric details,
evidence, practice, and the practice queue.
• Design and implementation: The task focus, layout, and user-facing English
copy reflect my design decisions. The clickable HTML was built with implemen-
tation support from Cursor (AI-assisted editing) as a medium-fidelity mockup,
consistent with course guidance that prototyping tools may be used while the
design remains my own.
Main flow (one line): Overview → Rubric breakdown → Row detail (including Evi-
dence) → Practice (State → Example → Match → Checklist) → Practice queue.
